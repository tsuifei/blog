About TsuiFei

一位在歐洲的大嬸，很久很久以前是平面設計師，2001 開始自學網站設計，從最基礎的 html 和 css 開始，期間也執行了不少網站（甚至是 flash’flash 啊！）工作模式大都以個人接案為主，八年前進入某家電腦周邊批發公司，管理三品牌網站與行銷、B2B 網站，電子報、並使用老本行技術，設計廣告、目錄、產品拍照、產品包裝，沒錯，就是一條龍的概念。
離開此間公司後申請進修成功，決定來個徹底技術大升級，轉職前端工程師。

熟悉 HTML,CSS, Javascript 與框架。

專業簡介：

平面設計、網站設計、前端。

專業成就:

不小心轉行好幾遍：
轉「作家」出了三本旅遊書，其中一本 14 年還在再版，並且為國內設計相關雜誌撰稿多年。
自學轉執「網站設計師」，也這樣做了十幾年。從架設網站到建資料庫、電子報等行銷相關工具，全自學。
轉職「前端工程師」，人生總要翻轉才有趣。
那我的本行呢？是平面設計師啦！現在是前端
